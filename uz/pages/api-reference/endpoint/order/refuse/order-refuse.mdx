## Umumiy ko'rinish

**Order Refuse API** mavjud buyurtma uchun refund amaliyotini bajarish uchun ishlatiladi.

Bu endpoint fiskal qaytimni ro'yxatdan o'tkazadi, mahsulotlar summasini qayta hisoblaydi,
and returns fiscal information such as receipt number and QR code.

Avtorizatsiya **talab qilinmaydi**.
