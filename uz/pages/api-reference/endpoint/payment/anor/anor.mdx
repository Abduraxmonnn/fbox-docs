## Umumiy ko'rinish

**Anor Payment API** Anor orqali to'lovni boshlash uchun ishlatiladi.

Bu endpoint to'lov tranzaksiyasini yaratadi va `payment_id` qaytaradi
that must be used later for fiscalization.
