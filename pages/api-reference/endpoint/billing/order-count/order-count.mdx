## Overview

The **Orders Count API** is used to retrieve aggregated sales information
for a specific company within a selected time period.

This endpoint returns the total number of orders and the total sales amount
based on the provided filters.

---

## Authorization

This endpoint **requires authentication**.

**Header:**

Authorization: Token `<token_value>`


The token must be prefixed with the word `Token` followed by a space.
